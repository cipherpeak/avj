# Mine Diamond Necklace Set - Luxury E-Commerce Landing Page

A high-end, luxury e-commerce landing page for the Mine Diamond Necklace Set, built with React, Tailwind CSS, and Lucide-React icons.

## Design Aesthetic

- **Theme**: Minimalist luxury with Soft White, Charcoal Grey, and Champagne Gold accents
- **Typography**: Playfair Display (Serif) for headings, Inter (Sans-Serif) for body text
- **Layout**: Clean grid system with generous whitespace and smooth hover transitions

## Features

- **Sticky Navigation**: Minimalist header with logo, navigation links, and shopping bag icon
- **Product Showcase**: Hero section with large image placeholder and product details
- **Detailed Specs**: Two-column grid showcasing necklace and earring components with size and gold color options
- **Price Breakup**: Elegant accordion showing cost transparency (Gold, Diamond, Making Charges, Tax)
- **Trust Signals**: Icons for Certified Diamonds, Secure Checkout, and Free Insured Shipping
- **Responsive Design**: Fully responsive across mobile, tablet, and desktop
- **Sticky CTA**: Mobile-optimized sticky "Inquiry & Purchase" button

## Tech Stack

- **React 18** - UI library
- **Tailwind CSS 3** - Utility-first CSS framework
- **Lucide React** - Icon library
- **Vite** - Build tool and dev server

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173/`

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
